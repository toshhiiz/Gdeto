## ⏳ Что Нужно Сделать

### 1️⃣ Создать Backend

Используйте `BACKEND_API_SPEC.md` - там полная спецификация всех эндпоинтов.

**Stack**: Node.js + Express + MongoDB/PostgreSQL

**Основные эндпоинты**:
- `/api/auth/register` - регистрация
- `/api/auth/login` - вход
- `/api/properties` - список объявлений
- `/api/properties/:id` - одно объявление
- `/api/properties` - создание объявления
- `/api/favorites` - избранное

### 2️⃣ Запустить Frontend

```bash
cd gdeto-frontend
npm install
npm run dev
```

Откроется на `http://localhost:5173`

### 3️⃣ Запустить Backend

Когда бэкенд будет готов:

```bash
npm start
# Должен работать на порте 3000
```

### 4️⃣ Установить API URL

Создайте файл `gdeto-frontend/.env.local`:

```env
VITE_API_URL=http://localhost:3000/api
```

Готово! Frontend автоматически начнёт использовать ваш бэкенд.

## 📞 Помощь

### Frontend Ошибка?
1. Проверьте консоль браузера (F12)
2. Смотрите Network tab для API ошибок
3. Убедитесь что VITE_API_URL правильный

### Backend Ошибка?
1. Проверьте server logs
2. Убедитесь что БД подключена
3. Проверьте формат response (должен быть JSON)
4. Убедитесь что порт 3000 свободен
